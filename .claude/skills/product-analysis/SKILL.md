# 상품 분석 스킬

상품 기획 문서(.md, .docx 등)를 분석하여 마케팅과 개발에 필요한 구조화된 정보를 추출합니다.

## 분석 프로세스

### 1. 기획서 핵심 컨셉 추출
- 브랜드 철학/가치
- 포지셔닝 문장
- 핵심 메시지

### 2. 상품 구조 정리
- 레벨/등급별 분류
- 각 레벨의 특징
- 가격대 및 타겟

### 3. 타겟 고객 세그먼트
- 주요 타겟 정의
- 고객 페인포인트
- 기대 효과

### 4. USP 및 차별점
- 경쟁사 대비 우위
- 기술적 차별점
- 고객 관점 혜택

## 출력 포맷

분석 결과는 다음 형식으로 정리합니다:

```
## 브랜드 개요
- 브랜드명:
- 핵심 컨셉:

## 상품 라인업
| 레벨 | 특징 | 타겟 | 가격대 |

## USP
1.
2.
3.

## 기술적 차별점
-
```

## 의료/미용 분야 특수 고려사항
- 의료 용어는 일반인 친화적 표현으로 병기
- 수치 데이터 정확히 보존
- 과장 표현 피하기
- 의료법 준수 확인
