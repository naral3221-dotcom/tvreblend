---

## 1. 핵심 메시지

### 메인 슬로건
> **"직접 만든 다양한 실로, 내 얼굴에 딱 맞는 최적의 조합"**

### 서브 카피
- "투명하게 어려지고, 투명하게 오래가는"
- "당신의 얼굴을 설계하는 Designer & Maker"
- "기성품 리프팅이 해결 못하는 미세한 영역까지"

### 원 라인 피치
> 라포엘은 병원이 직접 만든 6종의 실로 당신만의 맞춤 리프팅을 설계합니다.

### Section 2: 문제 제기

**섹션 타이틀**
```
왜 기성품 실로는 부족할까요?
```

**본문**
```
모든 얼굴은 다릅니다.
피부 두께, 처짐의 정도, 원하는 변화까지.

하지만 대부분의 실리프팅은
정해진 기성품 실 하나로 모든 얼굴을 시술합니다.

내 얼굴에 딱 맞는 리프팅,
정말 불가능할까요?
```

**핵심 문제점 3가지**
1. 한 가지 실로는 다양한 얼굴형에 대응 불가
2. 초기 효과 vs 유지력, 둘 중 하나만 선택해야 함
3. 어떤 시술이 나에게 맞는지 알기 어려움

```

```

```
Designer & Maker, 라포엘
```

**본문**

라포엘은 기성품에 의존하지 않습니다.

당신의 얼굴 구조와 고민에 맞춘 최적의 조합을 찾습니다.

장실로 끌어올리고, 단실로 채우는 입체 설계.
PDO의 고정력과 PCL의 지속성을 동시에.

이것이 라포엘만의 맞춤 리프팅입니다.

**핵심 포인트 3가지**

| 🔧 | 자체 제작 | 병원이 직접 설계하고 만든 라포엘 실 |
| 🧬 | 6종 라인업 | 효과와 부위별 최적화된 실 구성 |
| ⚡ | 복합 소재 | PDO + PCL, 고정력과 지속성 동시에 |

## 3. 수가 방어 논리 (상담용)

### LEVEL 2 기준 (260만원 이상)

**핵심 메시지**
```
"단순히 줄 수를 늘리는 것이 아닙니다."
```

**전달 포인트**
1. 얼굴 전체의 밸런스를 위해 소재와 길이를 다르게 적용한 **복합 설계**
2. 장실로 큰 윤곽을 잡고, 단실을 촘촘하게 배치
3. **얼굴의 뼈대를 다시 세우는 과정**
4. 의료진의 숙련된 디자인 감각이 필수적인 **프리미엄 시술**

**가치 전달 프레임**
```
"줄 수가 아닌 설계의 차이입니다.
같은 재료라도 건축가가 설계하면 결과가 다르듯이,
라포엘은 의료진이 직접 만든 실로 당신의 얼굴을 설계합니다."
```

## 4. 톤앤매너 가이드

### 브랜드 보이스
- **신뢰감**: 의료 브랜드로서의 전문성
- **절제**: 과장 없이 설득력 있게
- **따뜻함**: 고객 관점에서 공감하며

### Do's
- 고객의 고민에 공감하는 표현
- 전문성을 쉽게 풀어 설명
- 명확하고 간결한 문장

### Don'ts
- 과장된 표현 ("최고", "최초", "완벽한")
- 불안감 조성하는 표현
- 지나치게 기술적인 용어만 나열

### 문장 스타일
- 짧은 문장 선호
- 단정적 종결 ("~합니다")
- 필요시 질문형으로 관심 유도
